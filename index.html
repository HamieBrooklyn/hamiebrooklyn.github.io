Getting started

Player guide

1. How you use the bot

  • Slash commands: type / and pick a command (for example /balance, /daily).
  • Text shortcuts (optional): many card commands also work with the prefix c (for example ccd for a drop) or by mentioning the bot first. If that does not work in your server, stick to slash commands.

2. First steps

  1. /balance — see your Pokedollars and Crystals.
  2. /daily — claim your daily reward when it is available.
  3. /cd or ccd — card drop: opens a pack-style pull; use the buttons to choose what you keep. Card IDs may appear as plain text under the message so you can copy them on mobile.
  4. /coll or ccoll — list saved cards (often with pages).

3. Collection & search

  • /cs — search the global catalog or your collection (pick the mode the command offers).
  • /cv — full card view (large image + details) for one card.
  • /colv — flip through your cards like a binder (arrow buttons).
  • Web collection — sign in with Discord and browse your whole binder in a grid, search by name, sort by rarity/HP/damage, and sell cards to the shop using the bot’s collection buyout pricing (quote_collection_sell_payout on a deployed bot from Poke-Cards 97c5d1f+).
  • Web auctions — browse and search live listings, bid in Pokedollars or Crystals, and create listings from cards you own.

4. Packs (10 cards + 1 code card)

  • /packv — find packs by series / set, browse art, and buy with Crystals on a series. Rarer packs cost more Crystals.
  • /packd — random pack for 5,000 Pokedollars or 10 Crystals (or your server’s shop consumable, if configured).
  • /packcolv — see unopened packs; use Open, flip through pulls, and Scrap cards you do not want (the rest stay in your collection).

Crystals come from voting (see /vote) and from code cards inside packs.

5. More features

  • /vote — vote rewards (for example Top.gg).
  • trade — player trades (for example trade offer).
  • /auction — list, bid, and search auctions.
  • duel / deck — PvP duels and your duel deck. Use the web deck editor for an easier experience.
  • /leaderboard — server or global rankings: strongest cards, tankiest cards, rarest cards, and top auction sales.

6. Tips

  • If a new command does not show up yet, global slash commands can take a little while to update on Discord after a bot update.
  • Use /help or chelp if your server exposes a command list.
